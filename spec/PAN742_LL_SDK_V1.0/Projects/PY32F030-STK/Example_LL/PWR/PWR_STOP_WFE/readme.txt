================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了通过执行WFE(wait for event)指令进入stop模式，使用GPIO事件唤醒

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. LED闪烁一次后进入stop模式；
3. 在PA06上产生一个上升沿，产生GPIO事件，程序退出stop模式；
4. LED指示灯闪烁；
