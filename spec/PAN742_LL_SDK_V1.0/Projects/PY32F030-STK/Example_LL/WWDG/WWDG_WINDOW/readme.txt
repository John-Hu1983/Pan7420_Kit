================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了WWDG的 窗口看门狗功能，配置WWDG的窗口上限（下限固定是0x3F），程序中
通过delay延时函数，确保程序是在WWDG计数窗口内进行喂狗动作，通过LED灯闪烁，可以判
断窗口内喂狗并未产生复位。

================================================================================
测试环境：
测试用板：PY32F030_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序并运行；
2. LED灯闪烁
